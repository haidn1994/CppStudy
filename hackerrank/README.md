## Day 2

삼항 연산자는 안쪽에 식(Expression)이 반드시 와야 한다.  
문(Statement)이 오게되면 제대로 된 평가를 할 수 없게 된다.  


예를 들어 Continue나 break같은 키워드는 Statement로 평가되는데, 삼항 연산자 내부에 들어갈 수 없어서 컴파일 에러를 일으킨다.
